# 🎥 VideoConnect

A real-time video conferencing web app built with the MERN stack,
WebRTC for peer-to-peer video streaming, and Socket.io for signaling.



---

## 🚀 Features

- Real-time peer-to-peer video and audio calls via WebRTC
- Instant room creation and joining
- Real-time signaling with Socket.io
- Responsive UI for desktop and mobile
- MERN stack architecture (MongoDB, Express, React, Node.js)

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js |
| Backend | Node.js, Express.js |
| Database | MongoDB |
| Real-time | Socket.io |
| Video/Audio | WebRTC |
| Deployment | Vercel |
